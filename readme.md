## Install Instructions

 > Install requires Neovim 0.9+. Always review the code before installing a configuration.

Clone the repository and install the plugins:

```sh
git clone git@github.com:quackzar/vim-dotfiles ~/.config/quackzar/vim-dotfiles
NVIM_APPNAME=quackzar/vim-dotfiles/ nvim --headless +"Lazy! sync" +qa
```

Open Neovim with this config:

```sh
NVIM_APPNAME=quackzar/vim-dotfiles/ nvim
```
